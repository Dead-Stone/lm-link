Install preview APK

  1. 1
    Open GitHub Releases and download lm-link-preview-*.apk.
  2. 2
    On your phone, open the file and tap Install.
  3. 3
    If prompted, allow installs from your browser (Install unknown apps).
  4. 4
    Launch LM Link — the in-app tutorial guides setup.

Package com.lmlink.android · Previews ship on GitHub first

What you need

  • Android phone or tablet (recent version recommended)
  • Same Wi‑Fi as your Mac or PC for remote chat
  • LM Studio 0.4+ on your computer (remote mode)
  • Optional storage for on-device GGUF models

Quick start

  1. 1
    On your computer: load a model → Developer → enable Local Network + CORS → Start Server.
  2. 2
    In LM Link: Settings → Connection → scan or paste http://192.168.x.x:1234/v1 → Test → Save.
  3. 3
    Open a chat and pick a model from the footer.
For maintainers & developers

Publish a preview:

  1. npm run build:apk:local → APK in builds/
  2. Create a GitHub Release (e.g. v1.0.0-preview.1) and attach the APK
  3. Share this install page with testers

Build from source: github.com/Dead-Stone/lm-link — npm install, npm run android
