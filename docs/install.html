LM Link for Android

Chat with LM Studio on your Mac or PC over Wi‑Fi — or run GGUF models on your phone.

Install

Get it on Google Play Download APK (GitHub Releases)

Google Play listing is rolling out. If the store link is not live yet, use the APK or email us to join the closed test.

Install the APK

  1. Download the latest preview APK from GitHub Releases.
  2. On your phone, open the file and tap Install.
  3. If Android asks, allow installs from your browser or file manager (Install unknown apps).
  4. Open LM Link from your home screen.

Package ID: com.lmlink.android

What you need

  • Android phone or tablet (recent version recommended)
  • Same Wi‑Fi as your Mac or PC for remote chat
  • LM Studio 0.4+ on your computer (for remote mode)
  • Optional: storage space for on-device GGUF models

After installing

  1. On your computer: load a model in LM Studio → Developer → enable Serve on Local Network and CORS → Start Server.
  2. In LM Link: Settings → Connection → scan the network or paste your server URL (e.g. http://192.168.1.5:1234/v1) → Test → Save.
  3. Open a chat and pick a model from the bar at the bottom.

The in-app tutorial walks through each step with illustrations.

Build from source

Developers can clone the repo and run npm install then npm run android or npm run build:apk:local.
